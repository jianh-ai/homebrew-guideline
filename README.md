## 2.使用
1.使用homebrew安装程序
`brew install`

2.使用homebrew卸载程序
`brew uninstall`

3.查询homebrew可用程序
`brew search`

4.查询homebrew已安装应用
`brew list`

5.查询homebrew版本
`brew -v`

---
## 3.homebrew的存储路径
`/opt/homebrew/Cellar`

一般来说直接访达是无法直接查询的，得通过“聚焦搜索”才能搜寻到
